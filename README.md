# DIWALI_SALES_ANALYSIS
A complete data analysis project on Diwali sales using Python. Includes data cleaning, EDA, customer segmentation, and business insights to understand buyer behavior and improve sales strategies.
It includes data cleaning, EDA, visualizations, and insights using Pandas, NumPy, Matplotlib, and Seaborn.
# Project Objectives
Understand which customer groups buy the most during Diwali

Identify top-performing product categories

Analyze purchase patterns by gender, age, occupation, income, and state

Provide business insights to increase revenue
